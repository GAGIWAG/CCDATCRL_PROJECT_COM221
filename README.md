**How to run**

*Prerequisite Libraries*
- customtkinter
- pyxl
- ctkmessagebox
- pillow
- pandas
- tkinter

*Library Installation*

copy paste the following in your cmd prompt

- pip install customtkinter
- pip install pyxl
- pip install ctkmessagebox
- pip install pillow
- pip install pandas
- pip install tkinter

Note:
tkinter and customtkinter only works on python versions 7 to 11
